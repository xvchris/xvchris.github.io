一、Zsh 环境配置

1. 安装 Zsh

sudo apt update
sudo apt install zsh -y

2. 设置 Zsh 为默认 shell

chsh -s $(which zsh)

注：执行后注销或重启终端即可生效。

3. 安装 Oh My Zsh

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Oh My Zsh 帮助管理主题和插件，安装后会自动切换到 Zsh。
